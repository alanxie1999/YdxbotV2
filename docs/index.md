# YdxbotV2 Wiki

YdxbotV2 是一个基于 Telegram 的多账号自动化脚本，当前文档站主要服务于：

- 新 VPS 上的快速部署
- 账号配置与命令使用
- 日常运行、更新和排障

## 适合谁看

- 第一次部署脚本的人
- 需要维护账号配置的人
- 需要日常查看状态、更新版本、排障的人
