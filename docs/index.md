# YdxbotV2 Wiki

YdxbotV2 是一个基于 Telegram 的多账号自动化脚本，适合长期在群里手动盯盘、手动押注的人，把重复操作、状态查看、风险提醒和多账号管理集中到一套统一流程中。

## 核心能力

- 自动接收盘口消息与结算消息
- 按预设参数自动下注
- 支持多账号独立运行
- 支持模型预测与统计兜底
- 支持连输告警、盈利暂停、炸号保护
- 支持模型异常探测、自动切换与自动恢复
- 支持管理员控制台与通知渠道分离
- 支持版本更新、回退与运行状态查看

## 适用人群

- 长期手动盯盘、重复下注的人
- 同时管理多个账号的人
- 希望把告警、暂停、恢复、统计查看收进统一控制面的人
