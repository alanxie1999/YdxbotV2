## 8. 命令响应消息

### 8.1 帮助信息

### 位置

`zq_multiuser.py` - `_build_help_card()` / `process_user_command()` 函数

### 格式

```
📘 脚本命令指南

结论：常用命令直接点击 `代码` 即可快速复制。
常用：`st` `status` `pause` `balance`

⚡ 基础控制（最常用）
• `/st [预设名]` 启动（带名切换并启动）
• `/status` 查看运行状态看板
• `/pause` / `/resume` 暂停/恢复下注
• `/balance` 刷新当前账户余额
• `/stats` 查看连赢、连输详细统计

💰 资金与阈值
• `/gf [金额]` 设置菠菜资金上限
• `/set [炸] [赢] [停] [盈停]` 核心阈值
例：`/set 1 1000000 3 2`
• `/wlc [n]` 连输相关阈值

🤖 模型与策略（进阶）
• `/model list` 查看可用模型
• `/model select [编号/ID]` 切换模型（支持编号）
• `/apikey show` 查看与管理当前密钥
• `/apikey set` / `/apikey add` / `/apikey del` 管理密钥
• `/ms [0/1/2]` 切换模式（0=反投, 1=预测, 2=追投）

📋 预设与测算（进阶）
• `/yss` 查看全部预设
• `/yss dl [名]` 删除预设
• `/ys [名称] [连续] [止损] [一输] [二输] [三输] [四输] [首注]` 新增或覆盖预设
• `/yc [名]` 或 `/yc [参数...]` 按预设或临时参数测算

🛠 系统与数据（进阶）
• `/res tj` 重置收益/胜率统计
• `/res state` 彻底重置状态
• `/res bet` 只重置当前倍投链
• `/ver` 查看版本
• `/restart` 重启程序
• `/update [版本]` 更新版本
• `/reback [版本]` 回退版本
• `/explain` 查看最近判断依据
• `/users` 查看多用户状态
• `/xx` 执行辅助数据操作
```

## 样式特点总结

### Emoji 使用

- 📊 统计/仪表盘
- 🎯 目标/押注
- 💰 金额/余额
- ✅ 成功/赢
- ❌ 失败/输
- ⚠️ 警告
- ⛔ 暂停/阻止
- ⏸️ 暂停状态
- ▶️ 恢复/运行
- 🚦 状态
- 📋 清单/预设
- 🤖 模型/AI
- 🧧 红包
- 🎉 庆祝/成功

### 格式特点

1. **标题**：使用 Emoji + 粗体（`**标题**`）
2. **分隔线**：使用 `———————————————`
3. **键值对**：统一使用中文冒号 `键：值`
4. **层级**：使用 `-` 表示列表项
5. **强调**：使用 Emoji 前缀
