# 通知与消息说明

这页说明脚本在运行中会发哪些消息，以及它们各自的用途。

## 一、管理员控制台消息

管理员控制台主要接收：

- `status`
- 命令执行结果
- 启动成功
- 更新 / 回退 / 重启结果
- 连输告警
- 连输结束
- 模型异常与恢复
- 资金暂停
- 盈利 / 炸号暂停

特点：

- 信息更完整
- 更适合直接操作和排查

## 二、通知渠道消息

通知渠道主要接收：

- 重点提醒
- 风险提示
- 启动成功
- 模型链异常
- 更新结果

特点：

- 更偏“提醒”
- 不负责命令交互

## 三、常见消息类型

### 1. 下注执行

说明当前这一手已经下出去，等待结算。

### 2. 押注结果

说明本局已经完成结算，并回写输赢与状态。

### 3. 连输告警

说明当前连输已经达到高关注阈值。

### 4. 连输已终止

说明这一轮回补链已经结束，并展示本轮净盈利。

### 5. 资金暂停

说明当前菠菜资金不足以覆盖下一手，脚本已自动暂停。

### 6. 模型恢复探测

说明当前模型链正在后台探测可用模型。

### 7. 脚本启动成功

说明当前账号已经真正完成启动并进入可用状态。

## 四、当前显示口径

当前版本里：

- 金额统一显示为万元
- 连输结束中的收益按“本轮净盈利”理解
- `status` 会同时显示运行状态、模型状态和版本提示
