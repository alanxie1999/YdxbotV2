# YdxbotV2

Ydxbot 是一个基于 Telegram 的多账号自动化脚本项目。

## 说明

本项目当前以开源形式提供，主要用于：

- 自动化流程研究
- 多账号架构实践
- Telegram 交互与通知机制测试
- 文档与运维流程演练
